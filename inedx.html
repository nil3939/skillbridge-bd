About the Role
Shohoz is looking for a motivated Junior React Developer to join our engineering team. You will work alongside senior engineers to build and maintain the web frontend for our ride-sharing and logistics platform — serving over 1 million active users across Bangladesh. This is an ideal role for a fresh CSE graduate who wants real production experience from day one.
Responsibilities
  • Build and maintain reusable React components for Shohoz web apps
  • Integrate REST APIs from the backend team using Axios
  • Write unit tests with Jest and maintain 80%+ test coverage
  • Participate in code reviews and daily stand-ups
  • Optimize components for maximum performance across browsers
  • Collaborate with UI/UX designers to translate Figma designs into code
Requirements
  • BSc in CSE or related field (fresh graduates encouraged)
  • Proficiency in React.js, JavaScript (ES6+), and Tailwind CSS
  • Understanding of RESTful APIs and async programming
  • Familiarity with Git and basic CI/CD pipelines
  • Strong communication skills in English and Bangla
Required Skills
React.js JavaScript Tailwind CSS REST API Git / GitHub Jest (Testing) Figma (Basic)
